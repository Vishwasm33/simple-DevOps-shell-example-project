# 🐧 Simple DevOps Shell Example Project

This repository demonstrates the fundamentals of **Shell Scripting in DevOps**. It contains simple Bash scripts that automate common system administration and DevOps tasks, helping beginners understand how shell scripting can improve productivity and reduce manual effort.

---

## 📖 About the Project

Shell scripting is one of the core skills required in DevOps. It enables engineers to automate repetitive tasks such as file management, process monitoring, system updates, backups, and application deployment.

This repository includes beginner-friendly shell scripts that demonstrate the basics of Bash scripting and Linux command-line automation.

## 🚀 Running a Shell Script

Make the script executable:

```bash
chmod +x script.sh
```

Run the script:

```bash
./script.sh
```

Or execute it using Bash:

```bash
bash script.sh
```

## 📚 Basic Shell Commands

| Command | Description |
|----------|-------------|
| `pwd` | Display current directory |
| `ls` | List files and directories |
| `cd` | Change directory |
| `mkdir` | Create a directory |
| `rm` | Remove files or directories |
| `cp` | Copy files |
| `mv` | Move or rename files |
| `cat` | Display file contents |
| `grep` | Search text in files |
| `find` | Search for files |
| `chmod` | Change file permissions |
| `echo` | Print text to the terminal |

---

## 📝 Shell Script Example

```bash
#!/bin/bash

echo "Welcome to DevOps!"

echo "Current User: $(whoami)"

echo "Current Directory: $(pwd)"

echo "Today's Date: $(date)"
```
